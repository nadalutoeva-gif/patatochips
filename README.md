# patatochips
Сайт про искусственный интеллект в веб-разработке разработанный с использованием фреймворка 98.css
Искусственный интеллект в веб-разработке

Ссылка на работающий сайт: [https://твой-юзернейм.github.io/ai-webdev/](https://твой-юзернейм.github.io/ai-webdev/)

Описание проекта
Информационный сайт из 8 страниц, посвящённый применению искусственного интеллекта в веб-разработке. Рассматриваются AI-ассистенты, генерация кода, интеграция языковых моделей (LLM) и облачные AI-сервисы. Проект стилизован в ретро-эстетике Windows 98 с использованием фреймворка 98.css.

Использованные технологии
- HTML5 — семантическая валидная разметка
- CSS3 — кастомные свойства, Flexbox/Grid, тёмная тема, микроанимации
- CSS-фреймворк — [98.css] (стиль Windows 98)
- JavaScript (ES6+) — нативное бургер-меню, переключение тем, форма подписки
- Деплой — GitHub Pages

Инструкция по локальному запуску
1. Клонируйте репозиторий: git clone https://github.com/твой-юзернейм/ai-webdev.git
2. Откройте index.html в браузере.
Или просто запустите локальный сервер (например, Live Server в VS Code).

Структура проекта
- index.html — Главная страница
- pages/ — Остальные страницы сайта
- css/custom.css — Кастомные стили и темизация
- js/main.js — Интерактивные элементы
